# Skylabs #3 - React workshops - 07.09.2018

## Polls App

1. Clone repository
    - `git clone https://github.com/skygate/skylabs.git`
2. Get into project directory
    - `cd skylabs`
3. Install dependencies
    - Yarn: `yarn`
    - NPM: `npm install`
4. Start dev server
    - Yarn: `yarn start`
    - NPM: `npm run start`
